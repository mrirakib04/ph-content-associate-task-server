# Server - PH Newspaper - Content Associate Task

The PH Newspaper backend is built using **Node.js**, **Express.js**, and **MongoDB**.
It provides RESTful APIs for fetching the latest news, retrieving individual news details, and managing news data.  
The server is optimized for modern frontend frameworks like **Next.js**.
